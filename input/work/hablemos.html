Hablemos de México

I know, these days wordpress is not the best solution to make a blog, actually is not the best CMS (talking about performance), but this year I made this website for a company that makes tequila and i really love it, i mean wordpress is not fast as I would like but it has all the tools for managing the content

I did not designed the UI or even choose the color palette but I proposed the animations and interactions, off course the website can be better, it will be next year ;)

All the post are called by ajax, this can be bad for SEO, but if you solved with changing address and push events on GA, users can get a great experience with no SEO problems, to make comments the website have disqus a really cool and easy comments system
